# Introduce
starseesaw is a typeface developed by applying Armin Hofmann’s graphic systems. Characterized by its geometric, "non-square" (Tal-nemokkol) structure based on the Sebulsik system, the font consists of a total of 11,332 glyphs, including Korean, Latin characters, numbers, punctuation marks, and symbols. A defining aesthetic feature is the sparkling effect created at the points where strokes meet and narrow. The typeface is composed of six distinct structures. Designed as a modular (combinatorial) system based on Hofmann’s principles, each structure features unique heights and widths, offering a dynamic typographic rhythm.
